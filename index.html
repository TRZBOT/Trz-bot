Available Commands

  • add <name> <response> - Add a new command
  • edit <name> <new_response> - Edit an existing command
  • delete <name> - Delete a command
  • list - List all commands
  • info <name> - Get command information
  • search <query> - Search commands
  • stats - Bot statistics
  • export - Export all commands
  • import <json> - Import commands from JSON
  • button <message> <button1> [button2] ... - Create clickable buttons
  • lock - Lock current channel
  • unlock - Unlock current channel
  • ban <@user> [reason] - Ban a user
  • kick <@user> [reason] - Kick a user
  • timeout <@user> <minutes> [reason] - Timeout user
  • unban <user_id> [reason] - Unban a user
  • addrole <@user> <role> - Add role to user
  • play <YouTube URL> - Play music
  • join - Join your voice channel
  • leave - Leave voice channel
  • pause - Pause/resume music
  • welcome <message> - Set welcome message
  • farewell <message> - Set farewell message
  • serverinfo - Show detailed server info
  • level [@user] - Check user level
  • rank [@user] - Same as level
  • levelconfig <enable/disable> - Toggle level system
  • levelrole <level> <role> - Set role rewards
